🗺️ Mapa Mental Final

Conectando todos os conceitos do processador com programação, Linux, sistemas operacionais e cibersegurança.

🧠 CPU
📦 Registradores
  • RAX, RBX, RCX, RDX (gerais)
  • RIP (Instruction Pointer)
  • RSP, RBP (Stack)
  • RFLAGS (status)
  • 64 bits cada (x86_64)
💨 Cache
  • L1: 32-64 KB, ~4 ciclos
  • L2: 256 KB-1 MB, ~10 ciclos
  • L3: 8-64 MB, ~30-40 ciclos
  • Cache Hit vs Miss
  • Localidade temporal/espacial
➕ ALU / FPU
  • Soma, subtração, comparação
  • AND, OR, XOR, NOT
  • FPU: números decimais
  • IEEE 754 ponto flutuante
⏱️ Clock
  • Frequência em GHz
  • Ciclos por segundo
  • Sincroniza operações
  • IPC: instruções por ciclo
🧬 Núcleos
  • Single-core → Multicore
  • Hyper-Threading / SMT
  • Paralelismo vs Concorrência
  • Threads (pthread)
🚌 Barramentos
  • Dados (transporta dados)
  • Endereços (posição na RAM)
  • Controle (sinais de comando)
  • Largura: 32-bit / 64-bit
📚 Stack
  • LIFO (último entra, primeiro sai)
  • Push / Pop
  • Stack Frame
  • RSP aponta o topo
  • Stack Overflow
🏔️ Heap
  • malloc, calloc, realloc
  • free (libera memória)
  • Alocação dinâmica
  • Memory Leak
🐧 Sistema Operacional
  • Processos e Threads
  • Escalonamento (CFS)
  • Troca de Contexto
  • User Mode / Kernel Mode
  • Syscalls
🔔 Interrupções
  • Teclado, Mouse, Disco
  • Rede, Timer
  • IDT (Interrupt Descriptor Table)
  • Salva/restaura estado
⚙️ Execução
  • Fetch → Decode → Execute
  • Pipeline (5 estágios)
  • Código C → Assembly → Binário
  • Compilador (gcc/clang)
🛡️ Cibersegurança
  • Buffer Overflow
  • Stack Canary
  • ASLR, DEP/NX
  • ROP Chains
  • Engenharia Reversa

🔗 Conectando com Outras Áreas

💻 Linguagem C

Compilação (preprocessador → compilador → assembler → linker), variáveis na memória, ponteiros e endereços, loops e funções mapeados para Assembly, convenções de chamada (RDI, RSI, RAX).

🐧 Linux

Syscalls como interface CPU↔SO, /proc e /sys expondo hardware, strace mostrando chamadas de sistema, processos como arquivos, permissões e modos de execução.

🖥️ Sistemas Operacionais

Escalonamento de processos, troca de contexto, memória virtual (MMU), proteção entre processos, drivers como ponte hardware↔software.

🌐 Redes

Pacotes de rede geram interrupções na CPU, processamento de protocolos consome ciclos, buffers de rede na memória, DMA (Direct Memory Access) libera a CPU.

🛡️ Cibersegurança

Buffer overflow explora stack, ASLR randomiza endereços, DEP protege execução, ROP reutiliza código, engenharia reversa analisa Assembly.

🔍 Engenharia Reversa

objdump e gdb analisam binários, radare2 e Ghidra desmontam código, registradores revelam estado do programa, syscalls mostram comportamento.
